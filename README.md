# Streamlit 대시보드 만들기

- 대시보드
    - 정보를 보여주는 화면 -> 판단을 도와주는 도구
    - 이 대시보드를 보는 사람이 누군가?
    - (내가 볼때) 어떤 판단을 할 것인가? (나)
    - (다른 사람) 어떤 판단을 하게 할 것 인가? 무엇을 판단하게 할 것인가?
    - 어떤 데이터 쓰지? KPI(핵심 지표)를 뭘로 잡지?
    - 필터 - 어떤 기준으로 나눠보게 하지?
    - 차트 - 어떤 패턴을 보여줄까? (눈으로, 설득력을 보강)
    - 설명 - 데이터에 대한 설명 + 데이터 이후의 액션을 하게 하기 위한 설명

- 초보적 단계 흐름
    - 데이터 불러오기 -> 표 불러오기 -> 필터 기준 설정 -> KPI 카드
    -> 차트 1개 -> 차트 여러개(2~3) -> 요약 텍스트(문장)
    -> 정리(레이아웃을 변경, 탭 컬럼 등)
